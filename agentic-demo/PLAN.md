# HR 관리자 대시보드 (CLI 데모)

## 목표
aind-hr-project REST API를 사용하는 *관리자용 웹 대시보드* 를 처음부터 만든다.
프론트엔드는 `web-dashboard/` 서브폴더에 생성한다.

## 화면 구성 (단일 페이지)
1. **직원 목록** — 테이블 형태, 이름 검색 (입력값으로 필터)
2. **부서 현황** — 부서별 직원 수 막대 차트

## 레이아웃
- 좌측 70% 직원 목록, 우측 30% 부서 차트 (반응형)
- 한국어 UI

## API
- `GET /v1/employees` → 직원 배열
- `GET /v1/departments` → 부서 배열
- 백엔드: `http://localhost:8080`
- 정확한 타입은 `curl -s http://localhost:8080/v3/api-docs` 로 확인

## 비목표
- 데이터 수정/삭제 기능은 이번 범위에 포함하지 않음 (조회 전용)
- 백엔드 코드 수정 없음
- 라우터/인증 없음
